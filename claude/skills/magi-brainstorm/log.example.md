# MAGI-BRAINSTORM 構想ログ

/magi-brainstorm の各構想を1件1行で記録する（記録規則は SKILL.md「構想ログ」参照）。
「その後」= 構想メニューがその後どうなったか（/magi に回した・採用・お蔵入り）。後日わかった時点で更新する。

> **このファイルは雛形（`log.example.md`）。** 実ログ `log.md` はマシンローカルの生ログで、
> Git 追跡しない（`.gitignore` 済み）＝私的な議題を公開リポジトリに載せないため。
> SETUP 時に `log.md` が無ければこの雛形からコピーして作られる（既存の `log.md` は上書きしない）。
