# 概念インベントリ（60項目）

英文法の概念を「前提の依存関係」つきで並べた台帳。カルテの ID はここを正とする。

使い方:
- **呼び名**列 = その概念が箱0〜1のユーザーに使う言い方。箱2〜3では「正式名（呼び名）」の併記、箱4では正式名のみ
- **前提**列 = この概念を教える前に箱3以上であってほしいID。前提が未定着なら、先に前提のほうを「きょうのポイント」にする（前提が崩れたまま上物を説明しても「説明されてもわからない」二次挫折を生むだけ）
- 「-」= 土台概念（前提なし。いつでも教えてよい）
- 急所文の選定・きょうのポイントの選定は「箱0〜2 かつ 前提が満ちている概念」を優先する（= 背伸びしすぎない一段上）

## 補足

- 1つの文に複数概念が現れるのが普通。「きょうのポイント」に選ぶのは誤読の原因になったもの最優先、次に「箱0〜2かつ前提が満ちている」もの
- ここにない細かい語法・イディオムは C58/C60 に寄せるか、概念登録せず解説だけする（カルテを60項目より細かくしない。細かすぎる台帳は更新が追いつかず死ぬ）
