# MAGI 決議ログ

/magi の各合議を1件1行で記録する（記録規則は SKILL.md「決議ログ」参照）。
「採否」= 使用者が決議を実際に採用したか。後日わかった時点で更新する。

> **このファイルは雛形（`log.example.md`）。** 実ログ `log.md` はマシンローカルの生ログで、
> Git 追跡しない（`.gitignore` 済み）＝私的な議題を公開リポジトリに載せないため。
> SETUP 時に `log.md` が無ければこの雛形からコピーして作られる（既存の `log.md` は上書きしない）。

**骨格凍結**: 決議 #1（2026-07-04）により、ログが5件たまるまで骨格（5機構成・討論方式）の
再設計は凍結。それまでの改修はプロンプト字句・集計規則の明確化のみ。
