# Fable 回答（原文ママ）

了解です。添付のリストにある設定を上から順に全部埋めます。用語はなるべくかみくだいて、「なぜその値か」を一言ずつ添えます。

## まず前提のおさらい（30秒）

- **プリセットA＝一生モノ**：単語帳・瞬間英作文。締切なし、ゆるく長く
- **プリセットB＝締切あり**：概念・Cloze・模試。試験日までに仕上げる
- FSRSとは「次にいつ復習すべきかを自動計算してくれる仕組み」。人間がいじるのは基本「目標正答率」だけでOK

## 共通設定（AもBも同じ値でいい項目）

| 設定 | 値 | なぜ |
|---|---|---|
| 復習枚数の上限 | 9999 | 復習を翌日に持ち越すとFSRSの計算が狂う。復習は必ず全部消化する前提 |
| 復習上限に関係なく新規を導入 | OFF | 復習が溜まってる日に新規まで足すと破綻するので、ブレーキとして |
| 上限をメインデッキから適用 | ON（デフォルト） | 親デッキの上限で全体をコントロールできて管理がラク |
| 習得ステップ | `1m 10m` | 新カードは「1分後→10分後」に確認して定着。それ以降はFSRSにお任せ |
| 配置順序 | 追加した順 | 教材の章の順番どおりに学べる |
| 再習得ステップ | `10m` | 間違えたカードは10分後に1回だけ確認すれば十分 |
| 新規カードを集める順序 | デッキ順 | 章の順番どおり |
| 集めた新規カードを並べる順序 | 集めた順 | 同上。兄弟カード（対のカード）は下の設定が勝手に別日に散らしてくれる |
| 新規カード表示のタイミング | 復習の後 | 「忘れかけの復習」が最優先。新規は余力でやるもの |
| 日をまたいだ習得中カードの表示 | 復習と混ぜる（デフォルト） | 特に変える理由がない |
| 復習カードを並べる順序 | 想起可能率の降順 | 「今ならまだ思い出せる確率」が低い＝忘れる寸前のカードから先に救出 |
| FSRS | ON | 前提どおり |
| FSRSパラメータ | 月1回「最適化」ボタンを押す | あなたの回答履歴から計算精度が上がる。押すだけ |
| パラメータ変更時に既存スケジュールも変更 | OFF | ONにするとその日の復習が一気に膨らむ。使うのは後述の1場面だけ |
| 信頼性チェック | OFF | 処理が遅いだけで普段は不要 |
| シミュレータ類 | 触らなくてOK | 実験的機能。遊びたいときだけ |
| 兄弟カードの非表示（新規・復習・習得中） | 3つともON | 双方向カードやClozeの2つ穴で「さっき答え見たじゃん」を防ぐ |
| 音声 | デフォルトのまま | 音声カードを使っていないので影響なし |
| カードの自動送り | すべてOFF（0秒） | 勝手に次へ進む機能。模試カードで誤操作事故のもと |
| 履歴欠落期間の推定正答率 / 指定日より前の復習を無視 / カスタムスケジューリング | 触らない | サボり期間が長い人や上級者向けの補正機能。今は不要 |

## プリセットで変える設定（ここが本体）

| 設定 | A：一生モノ | B：締切あり |
|---|---|---|
| **新規/日** | **20** | **逆算：未導入枚数 ÷（残り日数×0.7）**。例：900枚・残り120日→12枚/日 |
| （理由） | 数千枚の長期戦。続けられるペースが正義 | 「試験前の最後の3割の期間は新規ゼロで復習に専念」するための式 |
| **目標正答率** | **0.85** | **0.90**（試験1ヶ月前に**0.92**へ） |
| （理由） | 0.90→0.85に下げると復習量がガクッと減るのに、忘れる量はほんの少ししか増えない。コスパ最強帯 | 本番で取りこぼすコストが高い。直前は少し高負荷でも記憶を固める |
| **復習間隔の上限** | **36500日**（デフォルト） | **残り日数÷2**。月1で更新（例：120日前→60日、90日前→45日、30日前→14日） |
| （理由） | 締切がないので縛る理由ゼロ | どのカードも「遅くとも試験までの折り返し地点」で必ずもう1回顔を出す保証になる |
| **苦戦カード（leech）判定** | 8回で**非表示（保留）** | 8回で**タグのみ** |
| （理由） | 何度も間違える単語は「カードの作りが悪い」。捨てて作り直す方が速い | 試験範囲は捨てられない。タグを目印に問題文を書き直す |
| **タイマー表示** | OFF／最長回答**60秒** | **ON**／最長回答**90秒**、解答表示でカウント停止ON |
| （理由） | 単語に時間計測は不要 | 模試1問20〜40秒の時間感覚を体に入れる。90秒は計測が切れない保険 |
| **負担軽減日** | 土日を「少なめ」にしてもOK | 全曜日「標準」のまま |
| （理由） | 一生モノは休んでも死なない。継続優先 | 締切前に出題を減らすのは自分の首を絞めるだけ |
| **デッキ個別の追加設定** | なし | 模試デッキだけ「このデッキ」で新規10/日・復習50/日に絞る（1枚30秒×50枚≒25分で頭打ち） |

## 「既存スケジュールも変更」をONにする唯一の場面

Bで復習間隔の上限を絞ったとき（例：60日→45日）だけ、一度チェックを入れて保存します。こうすると「すでに遠い未来に飛んでいたカード」も手前に引き戻されます。その日の復習は一時的に増えますが、これは織り込み済みの痛みです。終わったらチェックは外してOK。

## 直近の実例：AZ-900（7/17、あと約2週間）

Bプリセットの考え方をそのまま当てはめると：復習間隔の上限は**7日**、新規の導入は**今週末で打ち切り**、最後の2〜3日は普段の復習＋総ざらい、という運用になります。設定はいじる箇所が「上限7日」だけで済むのが2プリセット構成の強みです。
